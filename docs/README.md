# 文档中心

本目录区分权威通用规范、展厅专属规范和项目使用说明，避免规则散落或在多个文件重复维护。

## 权威规范

- [生成式个人主页通用设计规范总索引](GENERATIVE_PORTFOLIO_DESIGN_SPEC.md)：产品、内容、结构、随机系统、视觉、交互、工程与验收规则的唯一权威入口。
- [`spec/`](spec/)：通用设计规范的 0–28 节分册；必须按总索引阅读。
- [`rooms/`](rooms/)：42 个展厅的独立设计规范目录。每个文件须按 `NN-style-id.md` 命名。

## 项目说明

- [English README](../README.md)：项目概览、运行、维护与部署。
- [中文 README](../README.zh-CN.md)：中文项目说明。

## 维护边界

- 通用规则写入 `docs/spec/`，不要复制到展厅文档。
- 展厅专属规则写入 `docs/rooms/`，不要集中到通用规范。
- 当前实现说明写入 README；若与权威规范有差异，必须明确标注，不得把现状描述成目标规范。
