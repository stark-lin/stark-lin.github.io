# 21. 性能规范

## 21.1 基础要求

* 不依赖大型框架；
* 不依赖第三方动画库；
* 不依赖图片素材；
* 首屏快速显示；
* JavaScript 失败时仍有基本内容；
* 不阻塞页面阅读；
* 不使用高耗能持续动画。

---

## 21.2 Canvas 与 SVG

允许使用：

* CSS；
* SVG；
* Canvas。

优先顺序：

~~~text
CSS
↓
SVG
↓
Canvas
~~~

只有在 CSS 或 SVG 无法合理实现时使用 Canvas。

Canvas 必须：

* 响应设备像素比；
* 限制刷新频率；
* 在不可见时停止；
* Reduced Motion 下停止；
* 不承担核心文本。

---

## 21.3 装饰元素数量

每个展厅应定义最大装饰数量。

移动端应减少：

* 几何体；
* 线条；
* 粒子；
* 阴影；
* 动态节点；
* SVG 复杂度。

禁止无限生成 DOM 节点。
